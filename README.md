# Instagram Login Page Clone

A responsive Instagram-inspired login page built using HTML and CSS. The project replicates the modern design and layout of Instagram's login interface while focusing on responsive web design principles.

## Features
- Responsive design
- Modern Instagram-inspired UI
- Clean and structured layout
- Image assets integration

## Technologies Used
- HTML5
- CSS3
